# IranMTA Frontend Dashboard

This project is a  **frontend implementation of the IranMTA user dashboard** , built with **React** and  **TypeScript** .

The UI is inspired by the main control panel used in IranMTA servers (GTA / FiveM / MTA) and focuses on a modern, game-style user experience with a clean and scalable codebase.
